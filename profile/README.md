**Lyra** is an open-source framework for inference-time coherence in language models. It measures the gap between what a model *could* say (reflexes, templates, surface behavior) and what it *should* say (the residual — genuine capacity after reflexes are removed).

Five reflexes. Any model. Zero install.

### Repos

| Repo | What it does |
|------|-------------|
| [**lyra**](https://github.com/awakenfyi/lyra) | The core protocol — five quietings, traffic light coherence scoring, 15 shadow patterns. Paste into any system prompt. |
| [**lyra-protocol**](https://github.com/awakenfyi/lyra-protocol) | Coherence gates for agentic pipelines. Observer layer that scores intent preservation at every handoff. Python SDK. |
| [**lyra-verb**](https://github.com/awakenfyi/lyra-verb) | Plugin framework for Claude — Write Beautifully, Coach Beautifully, Edit Beautifully, The Basis. Verbs, not agents. |
| [**lyra-marketplace**](https://github.com/awakenfyi/lyra-marketplace) | Claude Code plugin marketplace. Install Lyra or Lyra Guard with one command. |

### Install

```
/plugin marketplace add awakenfyi/lyra-marketplace
/plugin install lyra@lyra-labs
```

Or paste the [protocol](https://github.com/awakenfyi/lyra) into any system prompt — ChatGPT, Claude, Gemini, Llama. It works immediately.
